# Thesis result reproduction

This directory contains the code needed to reproduce, end to end, all results,
tables, and figures cited in the thesis: from the original PCAPs and CSVs of
each dataset through to the final metrics and graphical material for chapters 3
and 4.

## 1. Layout

```
code/
├── scripts/
│   ├── preprocessing/      data preparation scripts (PCAP → ML-ready CSV)
│   ├── experiments/        training and evaluation scripts
│   └── visualization/      aggregation and figure generation
└── output/
    ├── figures/            generated by the visualization script (created automatically)
    └── results/            generated by the experiment and visualization scripts (created automatically)
```

## 2. Data

The datasets (PCAPs and CSVs, several GB in total) **are not included in this
repository**. They must be placed in a directory with the following structure,
pointed to by the `TFG_DATA_DIR` environment variable:

```
$TFG_DATA_DIR/
├── raw_data/
│   ├── CIC-IDS2017/{PCAP,CSV}/
│   ├── CIC-IDS2018/CSV/
│   ├── InSDN2020/{PCAP,CSV}/
│   └── UNSW-NB2015/{PCAP,CSV}/
└── processed_data/          generated by preprocessing scripts (created automatically)
```

```bash
# Create and activate a virtual environment
python3 -m venv .venv
source .venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Point to the data directory
export TFG_DATA_DIR=/path/to/data
```

## 3. Execution order

### Step 1: Data preparation (`scripts/preprocessing/`)

Most compute-intensive (hours per dataset); **can run in parallel**:

```bash
python scripts/preprocessing/prepare_dataset_ids2017.py
python scripts/preprocessing/prepare_dataset_insdn2020.py
python scripts/preprocessing/prepare_dataset_nb2015.py
```

CIC-IDS2018 needs no preprocessing: experiments 2 and 3 read its CSVs directly.

### Step 2: Experiments (`scripts/experiments/`)

```bash
python scripts/experiments/validation_intra_ids2017.py
python scripts/experiments/experiment0_nb2015_cross_domain.py
python scripts/experiments/experiment1_nfstream_ids2017_to_insdn2020.py
python scripts/experiments/experiment2_cicflowmeter_baseline.py
python scripts/experiments/experiment3_cicflowmeter_fewshot.py
```

Experiments 2 and 3 do not depend on Step 1 (they read raw CSVs directly).

### Step 3: Final figures (`scripts/visualization/`)

Must run after all experiments:

```bash
python scripts/visualization/generate_thesis_figures.py
```
